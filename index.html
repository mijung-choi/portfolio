AR 수조 낚시 체험

AR + 실물 놀이의 결합으로 아이의 상상력을 자극하는 몰입형 경험

문제 인식과 기획 의도

기존 키즈카페 및 장난감은 몰입 시간이 짧고, 단순 반복적입니다.
아이들이 움직임, 수집, 반응에 열광한다는 인사이트를 바탕으로, 몰입형 AR 수조 놀이를 기획했습니다.

AR 수조

실제 물이 담긴 수조에 상·측면에서 디지털 물고기와 물결이 투사됩니다.

스마트 낚시 장치

아이들은 장치로 물고기를 잡으며 상호작용합니다.

디지털 도감

잡은 물고기는 태블릿에 자동 등록되며, 이름을 붙이거나 꾸밀 수 있습니다.

사용자 흐름

[탐색] 수조 속을 관찰 →
[상호작용] 장치로 조작 →
[피드백] 물고기 잡힘 →
[결과] 태블릿에 등록 →
[활용] 꾸미기/교환/도감화

차별성 & 기대 효과

  • 정적 놀이에서 동적 몰입 놀이로 전환
  • 아이의 수집욕, 상호작용 욕구를 모두 충족
  • 기술 기반 체험 콘텐츠로 교육성과 창의성 강화

현재 상태 & 다음 계획

어린 시절의 나를 마주하는 감성 심리 체험

✨ 프로젝트 개요

'나는 어떤 아이였을까?'는 사용자가 동화 같은 하루를 체험하며
자신의 어린 시절을 돌아볼 수 있도록 설계된 감성 성향 검사 콘텐츠입니다.

🎯 기획 의도

  • 바쁜 일상 속 잊고 지낸 '어린 시절의 나'를 재발견
  • 감정 회복과 자아 탐색을 위한 스토리 중심 경험

🧩 주요 기능

    • 아침, 유치원, 저녁으로 이어지는 하루의 여정
    • 말풍선 기반 선택지와 간단한 그림 그리기
    • 한 줄 요약, 키워드 분석, 맞춤 추천 결과

    🧠 기술 스택

    React
    프론트엔드 SPA 구성
    Tailwind CSS
    감성 UI 스타일링
    Node.js
    백엔드 API 처리
    Canvas API
    간단한 그림 그리기 기능

    🌱 기대 효과

    - 자아 이해와 정서적 힐링을 동시에 제공
    - 콘텐츠 기반 확장이 쉬움 (친구 모드 / 성장한 나 모드 등)
    - SNS 공유/기록 기능을 통해 바이럴 요소 확보

    현재 상태 & 다음 계획
